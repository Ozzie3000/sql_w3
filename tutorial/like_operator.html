LIKE operator

  • The LIKE operator is used after a WHERE clause.
  • It is used to search for a pattern in a column.
  • There are 2 operators
  • % the percent represent zero or more characters
  • _ the underscore represents a single character
  • Now get wild and make some combos!

  • LIKE = 'a%' means starts with a followed by 0 or more characters
  • LIKE = '%a' means starts with 0 or more characters and then followed by a or end in a if 1 or more characters up front.
  • LIKE = '%what%' means contains 'what'
  • LIKE = 'what' means find 'what'
  • LIKE = '_a%' means a is the 2nd character followed by 0 or more characters
  • LIKE = 'a_%_%' means starts with a andhas at least 3 character total.
